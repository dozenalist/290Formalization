# 290 Formalization

[Website](https://dozenalist.github.io/290Formalization/) <br>
[Dependency Graph](https://dozenalist.github.io/290Formalization/dep_graph_document.html) <br><br>

To make changes to the website, edit the `Chapter.tex` files in `blueprint\src` <br><br>

In order to configure the website and graph, I run `.\buildall.ps1` in terminal before committing, which is a custom command that does the following:
 - Runs `leanblueprint all` (builds the website)
 - Copies the contents of `blueprint\web` to `docs`, where the site is hosted from
 - Runs `fix_links.py` (fixes the broken links generated by `leanblueprint all`)

I'm pretty sure this command only works in powershell, so you might have to make some other command like it to update changes to the website
